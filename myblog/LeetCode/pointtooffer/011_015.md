## 剪绳子

![](./images/012.png)

```python
class Solution:
    def cuttingRope(self, n: int) -> int:
        maxnum = 0
        for i in range(2,n+1):
            length1 = n//i
            length2 = n//i + 1
            maxnum = max(maxnum, pow(length1,i-1)*(n-(i-1)*length1), pow(length2,i-1)*(n-(i-1)*length2))
        return maxnum
```

## 剪绳子II

![](./images/013.png)

```python
class Solution:
    def cuttingRope(self, n: int) -> int:
        maxnum = 0
        for i in range(2,n+1):
            length1 = n//i
            length2 = n//i + 1
            q1 = pow(length1,i-1)*(n-(i-1)*length1)
            q2 = pow(length2,i-1)*(n-(i-1)*length2)
            maxnum = max(maxnum, q1, q2)
            
        return maxnum%1000000007
```

## 二进制中1的个数

![](./images/014.png)

```python
class Solution:
    def hammingWeight(self, n: int) -> int:
        return bin(n).count('1')
```

## 数值的整数次方

![](./images/015.png)

```python
class Solution:
    def myPow(self, x: float, n: int) -> float:
        if n ==0:
            return 1
        elif n==1:
            return x
        elif n==-1:
            return 1/x
        else:
            return self.myPow(x,n%2)*self.myPow(x*x, n//2)
```
