## 连续子数组的最大和

![](./images/042.png)

```python
class Solution:
    def maxSubArray(self, nums: List[int]) -> int:
        maxsum = nums[0]
        res = maxsum
        for i in  nums[1:]:
            maxsum = max(maxsum+i,i)
            res = max(maxsum,res)
        return res
```
