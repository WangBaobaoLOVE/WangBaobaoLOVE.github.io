## 二进制数转字符串

![](./images/32.png)

```python
class Solution:
    def printBin(self, num: float) -> str:
        s = "0."
        n = 32
        while n:
            num *= 2
            r = num % 1
            p = int(num//1)
            if num%1 == 0:
                s = s+str(p)
                return s
            s = s+str(p)
            num = r
            n -= 1
        return 'ERROR'
```

## 整数转换

![](./images/035.png)

```python
class Solution:
    def convertInteger(self, A: int, B: int) -> int:
        return bin((A&0xffffffff)^(B&0xffffffff)).count('1')
```
