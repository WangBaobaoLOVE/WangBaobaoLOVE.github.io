## 跳水板

![](./images/069.png)

```python
class Solution:
    def divingBoard(self, shorter: int, longer: int, k: int) -> List[int]:
        if k == 0:
            return []
        if shorter==longer:
            return [shorter*k]

        val = []
        for i in range(k+1):
            val.append(shorter*(k-i)+longer*i)
        return val
```
