## 交换和

![](./images/078.png)

```python
class Solution:
    def findSwapValues(self, array1: List[int], array2: List[int]) -> List[int]:
        sum1 = sum(array1)
        sum2 = sum(array2)
        delta = sum2-sum1
        if delta % 2 != 0:
            return []
        
        delta = delta//2
        s2 = set(array2)
        for i in range(len(array1)):
            if array1[i]+delta in s2:
                return [array1[i], array1[i]+delta]
        return []
```
